Types and Symbols List¶

This page contains all the list of symbols for type.

In Cinema 4D, a symbol stand for a constant value which has a name.

Note

Unfortunately not all symbols are documented in Python.
If you spot a missing one, please contact us on Plugin Café support forum.
To see a symbol’s value, you can either write it into the Python Console or either drag and drop it into the Python Console. See Python Console.
