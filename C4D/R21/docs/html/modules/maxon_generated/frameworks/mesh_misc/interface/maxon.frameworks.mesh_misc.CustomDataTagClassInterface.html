maxon.frameworks.mesh_misc.CustomDataTagClassInterface¶

Description¶

maxon.frameworks.mesh_misc.CustomDataTagClassInterface uses maxon.frameworks.mesh_misc.MeshAttributeClassInterface as a base interface and allows to add information and functionality to the mesh attribute to be used as tag.

It mainly contains the information about the c4d.CustomDataTag registration, read/write, icons and the optional display implementation.
