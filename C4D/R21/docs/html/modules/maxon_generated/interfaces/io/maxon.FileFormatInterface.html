maxon.FileFormatInterface¶

Description¶

maxon.FileFormatInterface allows to implement and register file formats with its detection algorithm.
The maxon.FileFormats registry allows to register any format (e.g. FileFormats.Browsable, FileFormats.ImageJpg…).
