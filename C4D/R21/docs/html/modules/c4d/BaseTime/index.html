c4d.BaseTime¶

class c4d.BaseTime¶

A BaseTime represents a time value within Cinema 4d.

Here is an example how to use a BaseTime object.

import c4d

bc1 = c4d.BaseTime(5, 10) # reduced to 1/2
bc2 = c4d.BaseTime(5)     # 5/1
bc3 = bc1 / bc2
doc.SetTime(bc3)

See also

Time Management in Cinema 4D.
