c4d.Quaternion¶

class c4d.Quaternion¶

Has methods for conversion to and from a rotation matrix: GetMatrix/SetMatrix/SetMatrixNorm.

New in version R17.048.

Attributes

Quaternion.w¶

The rotation angle.

Type: float

Quaternion.v¶

The direction vector.

Type: Vector
