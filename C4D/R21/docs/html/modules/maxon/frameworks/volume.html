The Volume Framework¶

Description¶

The Volume frameworks allows to import and edit OpenVDB data.

Interfaces¶

frameworks.volume.GridAccessorCreationInterface Helper class to create the templated interface maxon.GridAccessorInterface.
frameworks.volume.GridAccessorInterface maxon.frameworks.volume.GridAccessorInterface allows to access volume data.
frameworks.volume.PointsToVolumeListInterface Interface to convert point position to a volume.
frameworks.volume.VolumeInterface([data]) maxon.frameworks.volume.VolumeInterface represents a core volume object in cinema.
