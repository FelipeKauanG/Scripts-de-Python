What was New in R13¶

In R13.058¶

  • Additions:
    • BaseContainer.GetClone()
    • BaseView.PW_S(), BaseView.WP_S(), BaseView.PW_W(), BaseView.WP_W(), BaseView.ProjectPointOnLine(), BaseView.ProjectPointOnPlane()
    • HandleInfo.radius, HandleInfo.center, HandleInfo.CalculateNewPosition()

  • Miscellaneous changes:
    • Gradient.InvertKnots() now returns a bool
    • BaseSelect.SelectAll() now works as expected

  • Changes in the documentation:
    • Handles are now working in the examples : see Py-DoubleCircle, Py-RoundedTube and Py-SpherifyModifier
    • New example added for TagData plugins: Py-LookAtCamera
    • Information on getting the generator in a Python effector
    • New section in Plugin Structure about C4DPL_RELOADPYTHONPLUGINS message : Python Plugins Reloading
    • Cleanup (typos and small fixes) and missing functions/methods parameters documentation added

In R13.051¶

  • MemoryFileStruct is now accepted by these functions:
    • BaseBitmap.InitWith()
    • GeClipMap.InitWith()
    • ColorProfile.OpenProfileFromFile()
    • ColorProfile.WriteProfileToFile()
    • c4d.documents.LoadFile()
    • c4d.documents.LoadDocument()
    • c4d.documents.MergeDocument()
    • c4d.documents.SaveDocument()

  • The following methods and functions were implemented in past releases but were not documented:
    • MoData.GetFalloffs(), MoData.GetCurrentIndex() and MoData.GetBlendID()
    • BaseObject.GetAndCheckHierarchyClone() and BaseObject.GetHierarchyClone()
    • c4d.storage.GeGetMovieInfo() and c4d.storage.GeIdentifyFile()

In R13.029¶

  • MemoryFileStruct is now accepted by these methods and functions:
    • BaseBitmap.Save()
    • MovieLoader.Open()
    • MovieSaver.Open()
    • c4d.gui.SaveShortcutSet()
    • c4d.gui.LoadShortcutSet()
    • c4d.storage.GeGetMovieInfo()
    • c4d.storage.GeIdentifyFile()
    • c4d.storage.ReadHyperFile()
    • c4d.storage.WriteHyperFile()

  • Several changes in the render module:
    • Added optional argument of type BaseVolumeData for ChannelData constructor
    • Added VolumeData.GetWeights(), VolumeData.CalcVisibleLight() and VolumeData.GetSmoothedNormal()

  • Gradient.InitRender() and BaseShader.InitRender() now take an InitRenderStruct parameter

  • More methods added:
    • ShaderData.FreeRender()
    • BaseContainer.GetContainerInstance()
    • NodeData.GetDEnabling(), NodeData.IsInstanceOf() and NodeData.GetDocument()

  • Added function ActiveObjectManager_SetObject()

In R13.016¶

Important Additions¶

  • MoGraph access : New module c4d.modules.mograph
  • Shader API : New module c4d.modules.render and plugin data class ShaderData
